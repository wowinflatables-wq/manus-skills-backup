# Manus Skills Backup — Wow Inflatables

This repository is a backup of all custom Manus AI agent skills built for **Wow Inflatables** (wowinflatables.biz / wowjumpparty.com), a bounce house and party rental business serving Middle Georgia.

## Repository Structure

```
/skills/                              ← All 15 Manus skills (one folder per skill)
  bounce-safe/                        ← Safety, Cleaning & SOP Keeper
  competitor-watchtower/              ← Pricing & Inventory Intel
  content-cannon/                     ← A Month of Posts, One Shot
  damage-deposit-adjudicator/         ← Fair Assessment & Documentation
  event-opportunity-radar/            ← Local Event Prospecting
  five-star-flywheel/                 ← Review & Reputation Engine
  inventory-iq/                       ← Buy the Right Unit Next
  lead-winback/                       ← Recover Cold Quotes
  package-prize/                      ← Build & Price Rental Packages
  permit-coi-concierge/               ← Venue Compliance Made Easy
  quote-rocket/                       ← Inquiry-to-Quote in Minutes
  rebooking-radar/                    ← Annual Repeat-Customer Reactivation
  route-boss/                         ← Delivery, Load & Route Planner
  search-and-destroy/                 ← B2B Opportunity Finder & Outreach
  weather-guard/                      ← Forecast Watch & Reschedule Manager
```

Each skill folder contains a `SKILL.md` file (the core skill instructions) and any supporting assets or reference files.

## Skills Overview

| Skill | Description |
|---|---|
| **Bounce Safe** | Generates and maintains safety, cleaning, and SOP documents — inspection checklists, setup/teardown SOPs, attendant training, and incident forms. Grounded in ASTM F2374 and CPSC guidance. |
| **Competitor Watchtower** | Runs a recurring sweep of named local competitors — pricing, unit availability, new inventory, and active promos — and reports gaps and moves worth making. |
| **Content Cannon** | Generates a full month of themed, ready-to-post social media content — captions, hooks, hashtags, a dated posting calendar, and visual direction. |
| **Damage Deposit Adjudicator** | Turns post-event return photos into a fair, documented damage assessment plus an internal repair ticket and a professional customer-facing damage notice. |
| **Event Opportunity Radar** | Scans local calendars — festivals, school and church events, grand openings, HOA schedules — inside the service radius and surfaces ranked date-anchored booking opportunities. |
| **Five-Star Flywheel** | Builds reviews and manages online reputation the FTC-compliant way. Times post-event review requests and drafts professional responses to every review. |
| **Inventory IQ** | Analyzes bookings to recommend which inflatable to buy (or retire) next, with ROI and payback math. |
| **Lead Winback** | Recovers quotes that never converted with a timed, personalized multi-touch win-back sequence (text + email) that handles objections and offers a re-quote or sweetener. |
| **Package Prize** | Builds irresistible, ready-to-sell rental packages — priced right and paired with a dynamic, eye-catching ad. |
| **Permit & COI Concierge** | Researches permit, insurance, power, and anchoring rules for any delivery address before an event, then generates the COI request and a venue-ready compliance packet. |
| **Quote Rocket** | Turns any incoming rental inquiry into a fast, accurate, ready-to-send quote with delivery, add-ons, and deposit itemized. |
| **Rebooking Radar** | Wins repeat bookings by nudging last year's customers before their annual occasion comes around again. |
| **Route Boss** | Plans and optimizes a day's party-rental deliveries and pickups — route order, truck loading, and printable driver run sheets. |
| **Search and Destroy** | Finds and pursues B2B rental opportunities (VBS, school carnivals, corporate picnics, etc.) — qualifies leads and drafts outreach. |
| **Weather Guard** | Watches the weather for every upcoming booked event, classifies wind/rain risk, and drafts safety-first reschedule messages. |

## How to Use

Each skill folder contains a `SKILL.md` file with the full skill instructions. To reinstall a skill into Manus, copy the folder into your Manus skills directory or upload it as a `.skill` bundle (ZIP of the folder) to the Manus Skill Hub.
