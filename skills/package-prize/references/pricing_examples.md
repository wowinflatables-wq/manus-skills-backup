# Pricing Examples for Package Prize

This document provides worked pricing examples by package type, showing the full math from component value to final recommended price.

## 1. The "Backyard Birthday Bash" (Value Bundle)
*   **Items:** Standard Bounce House ($150) + 2 Tables/12 Chairs ($30) + Cotton Candy Machine ($50)
*   **Component Value (S):** $230
*   **Cost Floor (C):** Delivery/Setup ($40) + Labor/Fuel ($20) + Consumables ($10) + Wear ($15) = $85
*   **Pricing Posture:** Value Bundle (Target ~15% discount off à-la-carte)
*   **Math:** $230 * 0.85 = $195.50
*   **Demand Adjustment:** Standard weekend (no change).
*   **Final Recommended Price:** $199
*   **Value Framing Line:** "$230 value — yours for $199!"
