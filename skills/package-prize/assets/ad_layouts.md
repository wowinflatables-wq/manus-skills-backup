# Ad Layout Briefs for Package Prize

Use these layout guidelines when generating images or providing a design brief for the user.

## 1. Square (1080x1080) - Instagram/Facebook Feed
*   **Visual Hierarchy:** Image -> Headline -> Offer -> CTA
*   **Text Placement Zones:**
    *   Top 20%: Bold Hook Headline (e.g., "SUMMER BASH PACKAGE").
    *   Middle 60%: High-quality image of the main package items (e.g., bounce house and concession machine) in a relevant setting (e.g., sunny backyard).
    *   Bottom 20%: Price/Offer line, CTA, and Logo.
*   **Brand Element Positioning:** Logo in the bottom right corner. Brand colors used for text backgrounds or borders.
*   **Design Note:** Keep text concise; rely on the caption for details.

## 2. Vertical (1080x1920) - Stories/Reels/TikTok
*   **Visual Hierarchy:** Hook -> Image -> Details -> CTA
*   **Text Placement Zones:**
    *   Top 15%: Hook Headline (leave safe zone at the very top for app UI).
    *   Middle 50%: Vertical-friendly image or short video clip of the items in action.
    *   Lower Middle 20%: Bulleted list of included items (keep it brief).
    *   Bottom 15%: Price, CTA ("Swipe Up" or "Link in Bio"), and Logo (leave safe zone at the very bottom).
*   **Brand Element Positioning:** Logo centered at the bottom or top left. Use brand colors for bold text overlays.
*   **Design Note:** Must be eye-catching within the first second. Use dynamic angles.

## 3. Banner (1200x628) - Facebook Link / Website Header
*   **Visual Hierarchy:** Image (Left) -> Headline & Details (Right) -> CTA (Bottom Right)
*   **Text Placement Zones:**
    *   Left 50%: Strong, wide image showing the scale of the package.
    *   Right 50%: Headline, short description, Price/Offer, and a clear CTA button.
*   **Brand Element Positioning:** Logo top right or bottom right of the text section. Brand colors used for the CTA button.
*   **Design Note:** Needs to be legible when scaled down in a newsfeed.

## 4. Flyer (8.5x11) - Printable (Community Boards, Schools)
*   **Visual Hierarchy:** Headline -> Main Image -> Included Items -> Price -> Contact Info
*   **Text Placement Zones:**
    *   Top 15%: Large, attention-grabbing Headline.
    *   Middle 40%: Clear, high-resolution image of the package.
    *   Lower Middle 25%: Detailed list of included items and add-on options.
    *   Bottom 20%: Large Price, strong CTA, Phone Number, Website, and Logo.
*   **Brand Element Positioning:** Logo prominently at the bottom. Use brand colors for headers and borders to make it pop on a bulletin board.
*   **Design Note:** Include tear-off tabs at the bottom if appropriate, or a clear QR code linking to the booking page.
